
# Work flow
## Start Coding
```
git pull
```

## Push
```
git add --all
git commit -m "<message>"
git pull
git push (if not merge conflict)
```

## Merge Conflict
solve merge conflict
```
git add --all
git commit -m "<message>"
git push
```
